Widget de Breadcrumbs para Elementor com ícones, estilo personalizado e suporte a Schema Markup para SEO.

== Description ==

Exiba trilhas de navegação (breadcrumbs) com total controle visual através do Elementor. Perfeito para SEO e usabilidade!

**Recursos:**
- Totalmente integrado ao Elementor
- Suporte a Dashicons, Font Awesome e SVG
- Separador personalizável
- Controle de cores (ícone e texto)
- Suporte a Schema.org (SEO)
- Sem dependência de plugins adicionais

== Installation ==

1. Faça o upload da pasta do plugin para `wp-content/plugins/` ou instale via painel.
2. Ative o plugin.
3. No Elementor, arraste o widget "Breadcrumbs Custom" para seu layout.

== Frequently Asked Questions ==

= Ele funciona sem Elementor? =
Não. Este plugin é exclusivo para uso com Elementor.

= Ele adiciona Schema Markup? =
Sim, você pode ativar essa opção diretamente no widget.
